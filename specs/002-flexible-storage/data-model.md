# Data Model: Flexible Storage

## Entities

### JobArtifact

Represents a file generated by a pipeline execution.

| Field | Type | Description |
|-------|------|-------------|
| `job_id` | `str` | UUID of the job |
| `path` | `str` | Relative path within the job directory |
| `name` | `str` | Filename (e.g., "results.json") |
| `size_bytes` | `int` | File size in bytes |
| `artifact_type` | `enum` | `VIDEO`, `ANNOTATION`, `REPORT`, `LOG` |

### StorageConfig

Configuration for the storage subsystem.

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `root_path` | `str` | `./storage/jobs` | Base directory for all job data |
| `provider` | `str` | `local` | Storage provider type (only `local` supported now) |
